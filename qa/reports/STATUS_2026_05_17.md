## Implementation findings (new — flag for Phase 3 rewrite)

1. **No admin pagination exists.** `admin/src/app/patients/page.tsx` renders
   all rows client-side; backend `provider.service.getPatients` has no
   take/skip. No `PAGE_SIZE` constant anywhere → decision-6 floor = **30
   fillers**. The "page-2" rationale is moot; fillers serve the status /
   verification / open-alert / cross-practice **filter** cohorts only.
2. **Pre-existing journal-entry re-seed non-idempotency** (monolith bug):
   the journal upsert keyed on `measuredAt = daysAgo()` (Date.now-relative)
   never matched across runs → 65→130→…. Fixed (commit `9cfcaa2`,
   wipe+recreate like meds). STANDING REQ 3 now holds.
3. **`cd backend && npm test` is broken in this checkout** — npm-workspace
   hoisting puts jest at root `node_modules`; the script's literal
   `node_modules/jest/bin/jest.js` path fails. Pre-existing, repo-wide, NOT
   Phase 0. Working invocation: `cd backend && node --experimental-vm-modules
   ../node_modules/jest/bin/jest.js <pattern>`. The §L wording assumes a
   non-hoisted install. **Phase 3 flag.**
4. **resetUser does NOT "restore" — it WIPES.** TestControlService.resetUser
   deletes journal/alerts/notifications/escalation and PRESERVES the
   seed-stable identity rows (user/profile/medication/threshold/assignment);
   the *seed* restores dynamic state. The plan's §I "expect 4 alerts after
   reset" was inaccurate; the spec asserts the real contract (wiped→0, stable
   preserved, re-seed→restored). STANDING REQ 4 holds under this reading.
5. **§G audit changeType adapted.** `VerificationChangeType` has no `ALERT_*`
   members (alert lifecycle audit lives on DeviationAlert). The plan's
   ALERT_ACKNOWLEDGED/ALERT_RESOLVED audit rows were mapped to
   ADMIN_ASSIGNMENT_CHANGE / ADMIN_CORRECT (still 5 rows, ~14d span).
6. **§I spec is a heavy DB-integration test** (~16 min: 3× remote `runSeed`).
   Skips when `DATABASE_URL` absent. Emits a benign "Jest did not exit" open-
   handle warning (real PrismaClient pg pool) — tests pass 3/3.
